**Version:** 1.0

# Purpose

This document defines how every AI tool must participate in the development of the Triads Future website.

AI is a development assistant.

AI is not the decision maker.

All AI-generated output must follow the Brand System, Design Language, Experience Language, Component System, CMS System and Development System.

Documentation always wins.

---

# AI Philosophy

AI should accelerate development.

Never control development.

Every generated line of code should be

- Readable
- Maintainable
- Modular
- Commented
- Consistent

Never generate code simply because it works.

Generate code because it follows the project architecture.
