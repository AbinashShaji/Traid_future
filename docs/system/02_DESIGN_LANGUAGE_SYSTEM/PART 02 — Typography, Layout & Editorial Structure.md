# Purpose

Typography and layout are the foundation of every page.

Visitors will spend more time reading than interacting.

The entire interface must therefore be designed around comfortable reading.

Design should never compete with content.

Content is the hero.

---

# Typography Philosophy

Typography is the strongest visual element in Triads Future.

Not animations.

Not colours.

Not illustrations.

Typography should communicate confidence.

A visitor should understand the hierarchy before reading a single sentence.

---

# Typography Personality

The typography should feel

- Calm
- Modern
- Premium
- Professional
- Editorial
- Human

Never

- Futuristic for the sake of being futuristic
- Decorative
- Playful
- Gaming inspired
- Sci-Fi
- Corporate

# External Spacing

Sections are separated using whitespace.

Never use

Heavy borders

Large coloured blocks

Decorative dividers

---

# Hero Layout

The Hero receives the largest amount of whitespace on the website.

It should immediately communicate confidence.

Never overcrowd the hero.

---

# CTA Layout

Primary CTAs deserve breathing room.

The amount of surrounding whitespace communicates importance.

---

# Grid Philosophy

The grid exists to organise.

Visitors should never consciously notice the grid.

They should only feel that everything belongs together.

---

# Desktop Grid

Desktop should use available width intelligently.

Do not place every element inside a narrow centred column.

Allow layouts to breathe.

Editorial layouts are preferred over rigid symmetry.

---

# Tablet Grid

Tablet is redesigned.

Not resized.

Some layouts become

Two columns.

Others become

Single column.

The decision depends on readability.

Not on mathematical scaling.

---

# Mobile Grid

Single column.

Always.

Never force desktop layouts onto mobile.

Redesign.

Do not compress.

Some sections are typography-first.

Some are photography-first.

Some are story-first.
