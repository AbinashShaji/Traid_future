# Design Manifesto

Triads Future is not building a website.

Triads Future is building confidence.

Visitors do not buy websites.

Visitors buy trust.

Every layout,

every image,

every interaction,

every animation,

every line of code,

must increase trust.

If something looks beautiful but reduces trust,

remove it.

---

# Typography

Typography carries the personality.

Not colours.

Not animations.

Typography must remain readable on every screen.

# Photography
