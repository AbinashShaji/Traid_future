# Purpose

This document defines the complete architecture of the Triads Future project.

Every developer and AI agent must follow this architecture.

The project must remain scalable, maintainable and understandable for years.

The architecture should never depend on one developer.

It should depend on documentation.

---

# Architecture Philosophy

The project is divided into independent modules.

Each module has one responsibility.

Modules communicate through clearly defined interfaces.

Never tightly couple features.

Every feature should be replaceable without affecting the rest of the application.
