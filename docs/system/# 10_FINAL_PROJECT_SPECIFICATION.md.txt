# Purpose

This document is the complete implementation specification for the Triads Future website.

All previous documents define individual systems.

This document combines them into one complete product specification.

Every AI tool and developer must use this document together with the previous documentation.

This document defines **what must be built**, **how it should behave**, **how it should be developed**, and **when each feature should be implemented**.
